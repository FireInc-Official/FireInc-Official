## 🏢 About Us

**FireInc** is a small indie studio founded and run entirely by teenagers. We build things that matter to us — and hopefully to you too. Our team collaborates across games, websites, and software applications, turning ideas into real, shipped products.

> *"We don't wait to grow up to start building."*

We believe that age is not a barrier to creating quality software. Every project we release is a statement: young developers can ship professional work.
